Ada dua file pada repository ini, RPSB.pas dan writer.pas, yang dibuat menggunakan bahasa pemrograman Pascal. Kedua file ini gratis untuk diambil dan dimodifikasi.
Kedua file ini merupakan program text-based game yang cara memainkannya adalah dengan menggunakan integer untuk memilih pilihan. RPSB merupakan nama dari game ini yang saya ambil dari Rock-Paper-Scissors Battle yang disingkat.
Rock-Paper-Scissors karena beberapa move di game ini memiliki sifat segitiga seperti batu-gunting-kertas dan Battle karena game ini terdiri atas pertarungan.

Installation Guide 	:	1. Download file RPSB.pas dan writer.pas.
						2. Siapkan Compiler untuk bahasa pemrograman Pascal, seperti FPC (Free Pascal Compiler) yang dapat diunduh pada https://www.freepascal.org/
						3. Compile writer.pas menjadi writer.exe terlebih dahulu dan jalankan.
						4. Setelah writer.exe dijalankan, maka akan muncul file ending.dat.
						5. Setelah dikonfirmasi bahwa ending.dat sudah muncul di folder yang sama, compile RPSB.pas menjadi RPSB.exe.
						6. Jalankan RPSB.exe dan nikmati gamenya.
						
Additional Notes	:	1. Game dimainkan dengan cara memilih pilihan menggunakan integer atau bilangan bulat dan menekan enter.
						2. Jangan pernah masukkan character selain integer atau bilangan bulat karena hal ini dapat menimbulkan runtime error pada program.
						3. Tutorial untuk game sudah ada di dalam program.
						4. Jumlah Player untuk Game ini adalah 1-2 orang.
